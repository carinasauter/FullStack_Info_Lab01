1. Change Directory: cd nameofsubdirectory2. Create a Directory: mkdir nameofdirectory3. Rename a Directory: mv oldName newName4. List the contents of a directory: ls5. Move a file to a new location: mv oldFolderName newFolderName6. Copy a file to a new location: cp -R file newLocation7. Search for a string in a file: grep -q [PATTERN] [FILE] && echo $?8. Access the shell of a remote machine: ssh remoteAddress9. Remove a directory folder: rm -rf directoryname
